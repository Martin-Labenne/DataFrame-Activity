Etude du nombre de naissances aux états unis dans les années 1969 à 1988.
Les données brutes sont stockées dans le fichier births.csv 
Après un preprocessing visant à retirer les données aberrantes 
(cf description dans le fichier texte "netoyage des données"), 
le but était d'extraire 3 dataset (annees 60, annees 70, annees 80)

Le script python chargé du preprocessing est transformation_csv.py.
Il se charge, entre autre, d'agreger les données et de les exporter en 3 csv differents 

Le Rapport est lisible sur Jupyter Notebook car bien plus simple, à mon gout,
à mettre en page sur ce software.

Bonne lecture.
